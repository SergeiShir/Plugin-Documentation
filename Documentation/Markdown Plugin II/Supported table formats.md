# Supported table formats
#markdown2 #tables

For tables with headings - use this syntax:
```md
| Heading | Heading | 
| ------- | ------- |
| Data    | Data    |
| Data    | Data    |
```

For tables without headings - use this syntax: 
```md
| Data    | Data    |
| Data    | Data    |
| Data    | Data    |
```
